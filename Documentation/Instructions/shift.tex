\subsection{Shift Left}
\begin{verbatim}
    shl (BASE, IMM)    ; Direct
    shl ((BASE, IMM))  ; Deferred
    shl ((BASE, IMM))+ ; Auto Increment
    shl ((BASE, IMM))- ; Auto Decrement
\end{verbatim}

\subsubsection{Encoding}
This instruction will use the ARI format. 
The operation field (OP) wil be $1011_2$

\subsubsection{Operation}
$AC \leftarrow ShiftLeft(Memory[EA])$
\par 
Shift the memory contents of the effective address (EA) left.
Latch the overflow into the link register (L).
Place the result into the accumulator (AC).
