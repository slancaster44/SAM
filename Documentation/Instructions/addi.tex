\subsection{Add Immediate}
\verb|addi DR,SR,IMM|
\subsubsection{Encoding}
This instruction will use the RRI format. 
The operation field (OP) wil be $0011_2$

\subsubsection{Operation}
$DR \leftarrow SR + SignExtend(IMM)$
\par 
Add the contents of the source register (SR) with the sign-extended immediate (IMM).
Latch the overflow into the link register.
Place the result into the destination register (DR).
