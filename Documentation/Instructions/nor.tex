\subsection{Not-Or}
\begin{verbatim}
    nor (BASE + IMM)    ; Direct
    nor ((BASE + IMM))  ; Deferred
    nor (+(BASE + IMM)) ; Auto Increment
    nor ((BASE + IMM)-) ; Auto Decrement
\end{verbatim}

\subsubsection{Encoding}
This instruction will use the ARI format. 
The operation field (OP) wil be $0110_2$

\subsubsection{Operation}
$AC \leftarrow \overline{AC \lor Memory[EA]}$
\par 
`Not or' the memory contents of the effective address (EA) from the accumulator (AC).
Place the result into the accumulator (AC).
